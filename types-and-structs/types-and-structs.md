A ```type``` in Fortran is analogous to a ```struct``` in C/C++.

```fortran
type MyDataStructure
    integer :: a, b, c
    real(4) :: x, y, z
end type
```

```cxx
struct MyDataStructure {
    int a, b, c;
    float x, y, z;
};
```
